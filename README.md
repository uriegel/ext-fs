# ext-fs
Extensions to nodejs's fs
## Build steps
In Command line, enter

```npm install```

### Debugging
Build addon with the following command:

```npm install --debug```

Open ```dev/index.js``` and type ```F5```

Open Visual Studio and attach to process ```node.exe```
